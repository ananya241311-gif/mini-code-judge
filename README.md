# Mini Code Judge

## Overview
I built this project to understand how coding platforms evaluate solutions using test cases.

## Features
- Runs user-defined functions against test cases
- Displays pass/fail results
- Handles execution errors

## Concepts Used
- Functions
- Input/output validation
- Basic system logic

## How to Run
1. Clone the repository
2. Run: python code_judge.py

## Why I Built This
I wanted to understand how platforms like Codeforces or LeetCode check solutions internally.
